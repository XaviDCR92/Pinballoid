const uint8_t* IOLoadFile ( const char *const  strFilePath,
size_t *const  peSize 
)

Loads a file with absolute file path indicated by strFileName.

File data is stored into a dynamically-allocated buffer (via calls to malloc()) which can then be handled by external modules.

Parameters
strFilePathAbsolute file path e.g.: "cdrom:\\DATA\\SPRITES\\TILESET1.TIM;1".
peSizePointer to size_t variable where file size will be stored (we all know C does not support multi-return, shamefully).
Returns
Address to a read-only buffer with file data if successful, NULL pointer otherwise.
peSize is assigned to actual file size in bytes if successful, IO_INVALID_FILE_SIZE otherwise.
