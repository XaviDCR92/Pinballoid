static void ISR_RootCounter2 ( void  )
static

Root counter 2 interrupt handler.

Root counter 2 counter value is increased at a rate of System clock / 8 (33.868800 / 8 = 4.2336 MHz).

Remarks
Executed under interrupt context on timer compare value reached (every 10 ms according to RCNT2_COMPARE_VALUE).
See also
TimersInit() for timer compare value configuration.
