◆ I_MASK

#define I_MASK   (*(volatile unsigned int*)0x1F801074)

Interrupt mask register.

According to NoCash specifications, I_MASK bits are structured as follows:

  • Bit 0 IRQ0 VBLANK (PAL=50Hz, NTSC=60Hz)
  • Bit 1 IRQ1 GPU
  • Bit 2 IRQ2 CDROM
  • Bit 3 IRQ3 DMA
  • Bit 4 IRQ4 TMR0 Timer 0 aka Root Counter 0
  • Bit 5 IRQ5 TMR1 Timer 1 aka Root Counter 1
  • Bit 6 IRQ6 TMR2 Timer 2 aka Root Counter 2
  • Bit 7 IRQ7 Controller and Memory Card
  • Bit 8 IRQ8 SIO
  • Bit 9 IRQ9 SPU
  • Bit 10 IRQ10 Controller - Lightpen Interrupt
  • Bits 11-15 Not used (always zero)
  • Bits 16-31 Garbage
